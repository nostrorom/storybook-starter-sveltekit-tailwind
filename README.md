# Starter Template for Storybook - SvelteKit - TailwindCSS

## Setup with

- Typescript
- ESLint
- Prettier
- Playwright
- Vitest
- Darkmode/Lightmode

## Usage

```bash
# create a new project in my-app
npx degit nostrorom/storybook-starter-sveltekit-tailwind my-app

# install
cd my-app
pnpm i

# start
pnpm run storybook
```
